Create shareable GIFs from any YouTube video directly in your browser with YTgify.

## Quick Start

1. Install YTgify from the [Chrome Web Store](https://chromewebstore.google.com/detail/ytgify/jhjdimdoghooebpklbfhpggnaakiollk) or [Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/ytgify/)
2. Open any YouTube video
3. Click the YTgify button or press Ctrl+Shift+G
4. Set your clip timing, resolution, and frame rate
5. Add text overlay (optional)
6. Wait for processing, then download

## Settings Overview

- **Resolution**: 144p to 480p (lower = smaller files)
- **Frame rate**: 10-20 FPS (15 is the sweet spot)
- **Duration**: Up to 30 seconds
- **Text overlay**: Optional custom text

## Tips

- Keep GIFs under 5 seconds for best sharing
- Use 360p for balance of quality and file size
- 10-15 FPS works for most content
- Ensure video is fully loaded before setting start time

## Troubleshooting

| Issue | Solution |
|-------|----------|
| Wrong start time | Wait for video to fully load |
| File too large | Reduce duration, resolution, or FPS |
| Button not showing | Refresh the YouTube page |
